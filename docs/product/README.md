필요한 제품 문서만 골라 읽기 위한 안내다.

## 읽기 기준

```text
AI 챗봇 작업: docs/product/chatbot/CHATBOT_COMMON_SPEC.md를 먼저 읽는다.
프롬프트 구조/문장 규칙 확인: docs/chatbot/CHATBOT_PROMPT_SPEC.md, docs/chatbot/CHATBOT_PROMPT_KO_SPEC.md를 추가로 읽는다.
앱 전체 상태 확인이 필요할 때만: CURRENT_SPEC.md를 추가로 읽는다.
```

지도, 커뮤니티 등 다른 기능 문서는 챗봇 앱 제어 연결에 필요한 범위에서만 읽는다.
